# mealie-sdk

Auto-generated SDK packages for the [Mealie](https://mealie.io) API.

## Packages

| Package | Language | Install |
|---------|----------|---------|
| [mealie-typescript](./mealie-typescript) | TypeScript/JavaScript | `npm install mealie-typescript` |

## How it works

SDKs are automatically regenerated from Mealie's OpenAPI spec via GitHub Actions. When the API changes, a new version is published.
